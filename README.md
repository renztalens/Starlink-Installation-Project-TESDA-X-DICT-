# 🌐 Starlink Installation Project (TESDA x DICT)

This project documents my hands-on experience as a **Starlink Installer** during my Computer System Servicing (CSS) training under TESDA, in partnership with DICT.

The initiative aimed to provide reliable internet connectivity to remote schools in **Abra, Northern Luzon**, helping bridge the digital divide in underserved areas.

---

## 🚀 Project Overview

- 📍 Location: Abra, Northern Luzon, Philippines
- 🏫 Deployment Areas: Remote schools
- 🤝 Organization: TESDA x DICT
- 📅 Date: February 2026

This project involved full-cycle deployment of Starlink internet systems, from site survey to final network configuration and troubleshooting.

---

## 🛠️ Responsibilities & Tasks

### 🔍 Site Survey & Planning
- Conducted on-site assessments to determine optimal installation locations
- Evaluated signal strength, obstructions, and environmental conditions

### 📡 Starlink Installation
- Installed and mounted Starlink dish following safety standards
- Ensured proper alignment for optimal satellite connectivity

### 🌐 Network Setup
- Configured routers and wireless access points
- Established reliable Wi-Fi coverage for school environments

### 🔌 Cabling & Infrastructure
- Performed LAN cable routing and management
- Ensured clean, organized, and secure connections

### ⚙️ Network Configuration
- Set up IP configurations and connectivity settings
- Optimized network performance for stability and speed

### 🔐 Security Implementation
- Applied basic firewall configurations
- Blocked access to pornographic and gambling websites to ensure safe school environment

### 🧪 Testing & Troubleshooting
- Tested internet connectivity and performance
- Diagnosed and resolved network issues on-site

### 👨‍💻 Technical Support
- Provided on-site support during and after installation
- Ensured successful deployment for end users (students and staff)
